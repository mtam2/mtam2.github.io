How to fix Error 50 when starting DOA on Windows 8.1

  1. Download links

    2. SQL SERVER 2012 EXPRESS Download link
    When you hit the download button, check the boxes of
    NU\x64\SQLEXPR_x64_ENU.exe
    ENU\x64\SqlLocalDB.MSI

  2. Confirm it is error 50
  3. Confirm you are have SQL Server 2012 Express installed
  4. Confirm you are have SQL Server 2012 Express LocalDB
  5. Close all running instances of DOA
  6. Run CMD as Administrator
  7. Type "cd c:\Program Files\Microsoft SQL Server\110\Tools\Binn"
    Or wherever you installed SQL server to
  8. Type "sqllocaldb.exe delete v11.0"
  9. Now run DOA
    User: Admin
    Password: Password
