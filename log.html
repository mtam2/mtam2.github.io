How to unfreeze an object in Ruby!

In Ruby, there exists a method freeze which causes the variable to be immutable. There does not exist a unfreeze so a method to bypass this is to clone the object. You can clone the object to itself to unfreeze the variable.

a = "String".freeze
a.frozen? #true
a = a.to_s #"String"
a.frozen? #false

How to add columns to an existing database in rails

While in PSQL console:

rails generate migration add[COLUMN_NAME||GROUP_NAME]To[TABLE_NAME] COLUMN_NAME:DATA_TYPE COLUMN_NAME:DATA_TYPE
rake db:migrate
rails generate migration AddEmailToUsers email:string
rake db:migrate

You can also add contraints like indexes

rails generate migration AddIdToUsers id:integer:index
rake db:migrate
